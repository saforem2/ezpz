# `ezpz.distributed`

- See [`ezpz/distributed.py`](https://github.com/saforem2/ezpz/blob/main/src/ezpz/distributed.py)

!!! tip "Simplified rewrite of `ezpz.dist`"

    `ezpz.distributed` is a clean, simplified rewrite of [`ezpz.dist`](dist.md).
    It preserves the full public API surface while eliminating dead code,
    module-level side effects, and redundant aliases. New code should prefer
    importing from `ezpz.distributed`.

::: ezpz.distributed
